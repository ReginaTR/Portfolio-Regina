MEU PORTFÓLIO

Projeto Cadastro Doações

Instituições de caridade cadastram-se para que produtos com maior necessidade sejam listados afim de que usuários possam realizar doações junto à instituição.

Sistema de Login, Autenticação, Cadastro de Usuários, Cadastro de Produtos e Filtros.

Ferramantas: Ruby on Rails, RailsUI, Postgres, Devise, Ransack, Github, Kanban.

Projeto Lead Score

Captação de Leads e Extração de Scores para Clientes.

Sistema de Login, Autenticação, Jobs em Segundo Plano, Extração e Envio de Arquivos CSV, Operação de CRUD para leitura e validação de arquivos JSONB e Filtros de pesquisa.

Ferramantas: Ruby on Rails, Postgres, Sidekiq, Redis, Devise, RailsUI, Ransack, GitHub, Kanban.

Projeto Task Manager

Gerenciamento de Tarefas criadas pelo Usuário. Integração microserviços: Autenticação, Notificação e Web Scrapping.

Sistema de Login, Autenticação de Usuários, Cadastro de Tarefas, Filtros, Notificações, Web Scrapping.

Ferramantas: Docker, Ruby on Rails, MySQL, Faraday, JWT, Sidekiq, Redis, RailsUI, Ransack.
